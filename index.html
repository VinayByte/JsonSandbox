Welcome to the JSON Sandbox Repository!

This is a public repository dedicated to mocking, creating, and testing JSON REST APIs using GitHub. You can access the repository here.

Anyone is welcome to collaborate with this repository by forking the repository, making changes, and creating pull requests. To start collaborating, fork the repository from the above link, make your changes, and submit a pull request.

The base URL for accessing the API is: https://vinaybyte.github.io/JsonSandbox
